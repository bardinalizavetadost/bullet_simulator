# Модели данных

## Основные классы данных

### 1. Vector (Вектор)
Базовый класс для представления векторных величин в физической модели.

**Поля:**
- `x` - компонента по оси X
- `y` - компонента по оси Y

**Методы:**
- `len()` - вычисление длины вектора
- `angleRad()` - угол вектора в радианах
- `angleDeg()` - угол вектора в градусах
- `plus(Vector)` - сложение векторов
- `minus(Vector)` - вычитание векторов
- `times(double)` - умножение вектора на скаляр

**Пример использования:**
```java
Vector velocity = new Vector(100, 50); // Скорость 100 м/с по X, 50 м/с по Y
double speed = velocity.len(); // Общая скорость ≈ 111.8 м/с
double angle = velocity.angleDeg(); // Угол ≈ 26.6°
```

### 2. Point (Точка)
Класс для представления координат в метрической системе.

**Поля:**
- `x` - координата X в метрах
- `y` - координата Y в метрах

**Методы:**
- `toScreenX()` - преобразование координаты X в экранные пиксели
- `toScreenY()` - преобразование координаты Y в экранные пиксели

**Особенности:**
- Учитывает прокрутку экрана при преобразовании координат
- Ось Y инвертирована (вверх положительное направление)

### 3. BirdState (Состояние пули)
Немутабельная запись (record) для хранения состояния пули в определенный момент времени.

**Поля:**
- `time` - время в секундах
- `position` - позиция (вектор)
- `velocity` - скорость (вектор)

**Назначение:**
- Хранение истории полета для возможности отката времени
- Восстановление состояния симуляции

## Конфигурационные данные

PhyConfig (Физическая конфигурация)

**Поля:**
- `selectedConfig` - выбранная предустановка боеприпаса
- `position0X`, `position0Y` - начальные координаты
- `launchAngleDeg` - угол запуска в градусах
- `temperatureC` - температура воздуха в °C
- `pressurePa` - давление воздуха в Паскалях
- `startEnergyJ` - начальная энергия в Джоулях
- `ballisticCoef` - баллистический коэффициент
- `massG` - масса пули в граммах
- `caliber` - калибр в миллиметрах

**Вычисляемые свойства:**
- `bulletCrossSectionalArea()` - площадь поперечного сечения
- `massKg()` - масса в килограммах
- `airDensityKgM3()` - плотность воздуха
- `startSpeedMS()` - начальная скорость в м/с
