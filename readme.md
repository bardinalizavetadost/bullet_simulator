# Проект
# Bullet flight model

Симуляция физики на примере разных объектов. Наглядное представление процессов, которые пугали в школею

**Упрощенная модель** - баллистика для шарика. 

**Сложная модель** - учет сопротивления воздуха, смещения траекторий, в зависимости от окружающего мира и формы пули.

# Управление

1. Спомощью мыши можно выбирать примерный угол полета
2. Чем дальше мышь от птицы- тем больше скорость, подобрано под размеры экрана
3. Ручной ввод параментров в панель на экране
4. Возможность увидеть траекторию до начала полета
5. **клик мышью** - запустить птичку в полет к мишеням
