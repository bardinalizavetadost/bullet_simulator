# Проект
Описание проекта «BallisticLab: симуляция баллистики с учетом внешних эффектов»
Проект написан на языке Java.
Проект предназначен для образовательных целей. Он помогает наглядно показать физику, которой обычно пугают школьников, и объяснить разницу между идеальной и реальной траекторией полета. Также симулятор может использоваться как инженерный прототип для примерной оценки полета при разных условиях.
В проекте реализованы две модели расчета траектории. Упрощенная модель предназначена для шарика и использует школьные формулы кинематики без учета сопротивления воздуха. Сложная модель разработана для пули и учитывает сопротивление воздуха, а также влияние температуры, атмосферного давления и формы пули на полет.
Управление в программе осуществляется двумя способами. Первый способ — с помощью мыши: пользователь может выбирать примерный угол полета, а начальная скорость определяется расстоянием от шарика до курсора мыши — чем дальше мышь, тем выше скорость. Второй способ — ручной ввод всех параметров (угол, скорость, масса, форма пули, температура, давление) через панель управления.
До начала полета программа отображает предварительную траекторию, что позволяет оценить результат выстрела. На экране одновременно рисуются две траектории: одна показывается полет с учетом сопротивления воздуха, а другая — идеальная парабола без сопротивления. Пользователь может вызвать отдельный график, показывающий разницу между этими двумя траекториями для наглядной демонстрации влияния сопротивления воздуха.
В сложной части симулятора можно выбирать разные типы пуль (например, шар, оживальная пуля, конус), и для каждого типа на экране отображается соответствующая картинка. Запуск полета происходит кликом мыши по объекту, после чего шарик или пуля летит к установленным мишеням.
Главным в проекте считается наглядность представления физических процессов. Важно, чтобы пользователь мог легко сравнить идеальную и реальную траектории и увидеть, как сильно сопротивление воздуха и внешние условия меняют результат. Также важна удобная система управления: интуитивный выбор угла и скорости мышью в сочетании с возможностью точного ручного ввода для продвинутых пользователей. Отдельное внимание уделяется визуальному оформлению — разные картинки для разных типов пуль и четкая цветовая маркировка траекторий. Еще одной важной частью является корректная работа сложной модели, где учитывается сразу несколько факторов: форма пули, температура и давление окружающей среды.
